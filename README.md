# Web Traffic Analytics Dashboard

## Project Overview

This project focuses on analyzing website traffic data to understand user behavior, website performance, and engagement patterns. The analysis was performed using Microsoft Excel through data cleaning, KPI calculations, Pivot Tables, charts, and dashboard creation.

The objective is to identify high-performing pages, user entry and exit points, engagement levels, and traffic trends to support data-driven decision-making.

---

## Dataset

The dataset contains website analytics metrics including:

- Website
- Year
- Page Path
- Page URL
- Page Views
- Unique Views
- Average Time on Page
- Entrances
- Bounce Rate
- Exit Rate (%)

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

---

## Data Cleaning Process

The following preprocessing steps were performed:

- Checked for duplicate records
- Verified missing values
- Validated numerical columns
- Reviewed dataset consistency
- Ensured proper formatting of traffic metrics

---

## Key Performance Indicators (KPIs)

The following KPIs were calculated:

- Total Page Views
- Total Unique Views
- Average Time on Page
- Average Bounce Rate
- Average Exit Rate

---

## Analysis Performed

### 1. Top Pages Analysis
Identified the pages receiving the highest number of page views.

### 2. Bounce Rate Analysis
Analyzed pages where users left without further interaction.

### 3. Exit Rate Analysis
Identified major website drop-off points.

### 4. Entrances Analysis
Determined the primary entry pages used by visitors.

### 5. User Engagement Analysis
Evaluated pages with the highest average time spent by users.

### 6. Year-wise Traffic Trend Analysis
Analyzed website traffic growth and changes over time.

---

## Dashboard Features

The dashboard includes:

- Total Page Views KPI
- Total Unique Views KPI
- Average Time on Page KPI
- Average Bounce Rate KPI
- Average Exit Rate KPI
- Top Pages Chart
- Bounce Rate Analysis Chart
- Exit Rate Analysis Chart
- Entrances Analysis Chart
- User Engagement Chart
- Year-wise Traffic Trend Chart

---

## Key Insights

- High-performing pages contributed significantly to overall traffic.
- Certain pages acted as primary entry points for visitors.
- High bounce rate pages indicated opportunities for improvement.
- Exit rate analysis highlighted important drop-off locations.
- User engagement varied significantly across pages.
- Traffic trends provided insights into website growth patterns.

---

## Recommendations

- Improve pages with high bounce rates.
- Optimize pages with high exit rates.
- Enhance low-engagement content.
- Promote high-performing pages.
- Improve navigation from major entry pages.
- Monitor traffic metrics regularly for continuous improvement.

---

## Project Outcome

The project successfully transformed raw website traffic data into actionable insights through Excel-based analytics and dashboard visualization. The findings help understand user behavior and support website optimization strategies.
